## Maximum Likelihood and Bayesian Methods

Maximum Likelihood and Bayesian Methods are the two methods predominantly in use today for the analysis of molecular data for inferring phylogenies. They are both based on the concept of likelihood, or the probability of observing the data given some model.

In this case, for Maximum Likelihood it is the probability of observing the sequence data given the model of DNA evolution and the topology. We donʻt start off knowing all of the parameters for the model of evolution, nor the topology, we fit the parameters to the data, and search over many many possible topologies until we find the maximum likelihood tree. So specifying the model ends up being choosing how many parameters are included in the model and for what blocks of the data.  In maximum likelihood, the "best" model is the one that provides the greatest probability of observing the data, given the model (including evolutionary model and topology).
